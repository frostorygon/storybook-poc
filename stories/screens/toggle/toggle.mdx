# Toggle Screen

**The primary entry screen for holding or unholding a card.**

The `holdcard-toggle-screen` displays the current status of the customer's card (Active or On Hold) along with a contextually aware action button. It receives its data from the parent flow and delegates actions via events.

---

## Usage Example

```js
import { html, LitElement } from 'lit';
import { HoldcardToggleScreen } from '@scope/holdcard-toggle-screen';

class MyFeature extends LitElement {
  static get scopedElements() {
    return {
      'holdcard-toggle-screen': HoldcardToggleScreen,
    };
  }

  render() {
    return html\`
      <holdcard-toggle-screen
        .cardStatus="\${this.card.status}"
        .accountHolder="\${this.card.holderName}"
        .maskedNumber="\${this.card.maskedpan}"
        .isLoading="\${this.isFetching}"
        .error="\${this.hasError}"
        @action="\${this.handleToggleAction}">
      </holdcard-toggle-screen>
    \`;
  }
  
  handleToggleAction(e) {
    const { action } = e.detail; // 'hold' | 'unhold'
    if (action === 'hold') {
      this.api.holdCard();
    } else {
      this.api.unholdCard();
    }
  }
}
```
