# Temporary Holdcard

Allows the customer to temporarily hold or unhold their debit/credit card
from the native banking app. The feature opens as a webview.

## Native Layer Args

This feature receives the following arguments from the native app layer:

| Arg              | Type     | Description                      |
|------------------|----------|----------------------------------|
| `cardId`         | `string` | The card identifier              |
| `cardStatus`     | `string` | Current status: `active` or `on-hold` |
| `accountHolder`  | `string` | Customer display name            |
