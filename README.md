# Storybook POC

A Storybook Proof of Concept showcasing a **Temporary Holdcard** feature built with [Lion Web Components](https://lion-web.netlify.app/) and [Lit](https://lit.dev/).

## 🚀 Live Demo

👉 **[View Storybook](https://frostorygon.github.io/storybook-poc/)**

## Structure

| Section            | Purpose                                                    |
|--------------------|------------------------------------------------------------|
| **Docs & Overview**| Introduction, flow diagram, and navigation guide           |
| **Flows**          | End-to-end user journey scenarios with interaction tests   |
| **Screens**        | Individual screen components with fixtures and docs        |

## Getting Started

```bash
pnpm install
pnpm storybook        # dev server on localhost:6006
pnpm build-storybook  # static build → storybook-static/
```
